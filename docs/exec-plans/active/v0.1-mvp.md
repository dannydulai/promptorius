## Remaining
- [ ] **Cache persistence**: load/save cache to XDG_CACHE_HOME between renders
- [ ] **Icon map**: load nerd font icon names -> unicode codepoints
- [ ] **Exec timeout**: enforce per-exec-call timeout (default 10ms)
- [ ] **Global timeout enforcement**: currently relies on rayon; add explicit deadline

## Decisions
- Using rayon for concurrency (not tokio) — prompt rendering is CPU-bound, not IO-bound
- Rhai AST disk caching deferred to post-MVP (in-memory caching per render is sufficient)
- Segment config passed via Rhai Scope (not a registered function) so scripts use `config.get("key")`
- Stdlib scripts embedded in binary via include_str! — no external files needed
